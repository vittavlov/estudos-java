# Projeto Fênix: Revitalizando um Positivo Mobo para Dev Java 🚀

Este repositório documenta minha jornada de aprendizado em programação, começando do hardware!

## 💻 O Desafio
Transformar um notebook de baixo custo (Intel Atom, 2GB RAM) em uma máquina de estudos produtiva.

## 🛠️ O que foi feito:
- *Upgrade de Armazenamento:* Instalação de um SSD Samsung.
- *Sistema:* Linux Mint Xfce com 5.9GB de Swap para multitarefa.
- *Ferramentas:* Geany (IDE leve), Java JDK e Git.

## 📚 Estudos Atuais
- Lógica de programação com Java.
- Versionamento de código com Git.
